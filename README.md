## Getting started

```sh
npm install
```

Initialize the database

```sh
npm run db:init
```

Initialize the seed

```sh
npm run seed
```

Initialize the admin

```sh
npm run cms_init
```

Start

```sh
npm start
```

You can also start it in watch mode with

```sh
npm run start:dev
```

Initialize drop the database

```sh
npm run db:down
```
